# Project Idea

## Project Name
Antigravity Chat

## Vision
Create a free, anonymous chat platform where anyone can express thoughts without creating an account.  
All conversations automatically disappear after 24 hours to protect privacy.

## Problem

Many people want to express thoughts, feelings, or ideas but hesitate because:

- fear of identity exposure
- social judgement
- permanent digital records
- complicated login systems

Most chat platforms store conversations permanently and require accounts.

## Solution

A temporary anonymous chat web application where:

- No login is required
- A temporary user ID is automatically generated
- Messages, files, and reactions disappear after 24 hours
- Users can freely express ideas or thoughts

The platform is designed for fun, expression, and casual conversation.

## Target Users

- Students
- People who want anonymous conversations
- Communities discussing ideas
- Casual users looking for temporary chat spaces

## Core Principles

1. Privacy First
2. No permanent data
3. Simple access
4. Fast communication
5. Anonymous interaction

## Key Features

### Anonymous Access
Users can join the chat instantly without signup.

### Temporary User ID
When a user enters the app, a random ID is generated.

Example:
user_8432
user_1049

### 24 Hour Data Deletion
All content disappears automatically:
- messages
- emojis
- images
- files
- user IDs

### Media Support
Users can send:
- text
- emojis
- images
- files
- GIFs

### Real-time Chat
Messages appear instantly for everyone in the room.

### Simple Interface
Minimal design focused on chatting.

## User Flow

1 User opens website

2 System generates temporary user ID

3 User enters chat room

4 User sends messages / emojis / files

5 Messages visible to others in real time

6 After 24 hours system automatically deletes all chat data

## Safety Considerations

- File size limits
- Basic content moderation
- Rate limiting to prevent spam

## Future Improvements

- Multiple chat rooms
- Voice chat
- Temporary group chats
- AI moderation
- Reaction system
- Message reactions
- Room expiration timers

## Why the Name "Antigravity"

Antigravity represents freedom from constraints.

Just like gravity pulls things down, social pressure and identity sometimes restrict expression.

This platform removes that weight, allowing people to freely share thoughts.
